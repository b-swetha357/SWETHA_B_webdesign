# Professional Student Web Page – HTML5 & CSS3

## Project Overview
This project is a responsive professional student portfolio webpage developed using HTML5 and CSS3. The website follows semantic layout principles and is validated using W3C validation tools for standards compliance.

## Features
- Semantic HTML5 structure (header, nav, section, article, footer)
- External CSS3 styling
- Responsive design using media queries
- Optimized profile image (≤20KB)
- W3C HTML & CSS validated

## Technologies Used
- HTML5
- CSS3
- Git & GitHub
- W3C Validators

## Responsiveness
The webpage adapts to different screen sizes including desktop, tablet, and mobile devices using viewport meta tag and CSS media queries.
